# 🧠 Sistema de Estudo Inteligente

Projeto web desenvolvido para ajudar no aprendizado mais rápido e eficiente, utilizando conceitos de prática ativa e organização dos estudos.

---

## 💡 Sobre o projeto

Este projeto foi criado com o objetivo de explorar formas mais eficientes de aprendizado, mostrando como estudar melhor através de métodos práticos.

A proposta é apresentar um sistema simples que ajude o usuário a:

* Organizar seus estudos
* Praticar conteúdos
* Melhorar a retenção de conhecimento
* Aprender de forma mais ativa

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## 🧠 Ideia principal

O projeto se baseia no conceito de que aprender não é só consumir conteúdo, mas sim:

* Praticar
* Testar
* Revisar

Métodos como criação de testes e revisão ativa são amplamente usados para melhorar a memorização e o desempenho nos estudos.
